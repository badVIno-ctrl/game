# FPS ARENA

Онлайн PvP-арена + игра с ботами. Backend: FastAPI + WebSocket.

## Запуск локально

```powershell
# 1. Установить зависимости (Python 3.10+)
pip install -r requirements.txt

# 2. Запустить сервер
python main.py
```

Сервер поднимется на `http://0.0.0.0:8000`.

- На своей машине заходи: `http://localhost:8000`
- С другого устройства в ТОЙ ЖЕ Wi-Fi сети: `http://<IP_твоего_ПК>:8000`
  (узнать IP: `ipconfig` → IPv4-адрес, например `192.168.1.42`)
- Если Windows блокирует — разреши Python в Брандмауэре (всплывёт окно при первом запуске).

## Почему "не находит" в другом браузере

Если "другой браузер" — другое устройство:
1. оно должно быть в той же сети, что и твой ПК (или ты должен захостить игру в интернете);
2. сервер должен слушать `0.0.0.0` (уже исправлено в `main.py`);
3. порт 8000 должен быть открыт в брандмауэре.

Чтобы играть с другом из любой точки — нужен хостинг (см. ниже).

## Бесплатный хостинг

Так как нужен полноценный backend на Python с долгоживущими WebSocket-соединениями,
**не подойдут**: Cloudflare Pages, GitHub Pages, Vercel, Yandex Cloud Functions
(они для статики/коротких serverless-вызовов, а тут постоянные WS-сессии и in-memory state).

**Подходят (из твоего скрина — раздел "Полноценный backend"):**

### 1. Render.com (рекомендую)
- Бесплатный план: 512 MB RAM, "засыпает" после 15 мин неактивности (просыпается ~30 сек).
- WebSocket работает из коробки.
- Шаги:
  1. Залей проект на GitHub.
  2. На render.com → New → Web Service → подключи репо.
  3. Build Command: `pip install -r requirements.txt`
  4. Start Command: `uvicorn main:app --host 0.0.0.0 --port $PORT`
  5. Plan: Free. Deploy.
- Получишь публичный URL вида `https://fps-arena.onrender.com`.

### 2. Fly.io
- Бесплатный кредит ($5/мес), требует карту при регистрации.
- Не засыпает. WebSocket работает.
- Шаги: установи `flyctl`, в папке проекта `fly launch` → `fly deploy`.

### 3. Railway.app
- Триал-кредиты, потом платно. WebSocket ок.

### Для РФ без VPN
- **Amvera** (amvera.ru) — российский PaaS, есть бесплатный тариф, поддерживает Python + WebSocket.
- **Timeweb Cloud Apps** — тоже подходит (минимальный платный, но дёшево).
- Render/Fly из РФ открываются без VPN с переменным успехом — у пользователей игры может быть нестабильно.

## Структура проекта

- `main.py` — FastAPI сервер + WebSocket лобби/игровые комнаты
- `index.html` — главное меню (выбор режима, лобби PvP)
- `game.html` — сама игра
- `requirements.txt`, `Procfile` — для деплоя
